# Algoritms

Репозиторий с лабораторными работами и домашними заданиями по теме **алгоритмов и структур данных**.
Задачи сгруппированы по папкам (по номерам лабораторных) и покрывают базовые и продвинутые темы:
сортировки, поиск, работа со строками, жадные алгоритмы и динамическое программирование.

## Стек
- Python (3.x)

## Структура репозитория

- **Лаба 1 (матрицы)** — задачи на работу с матрицами
- **Лаба 2 (словари)** — задачи на словари / отображения
- **Лаба 3 (сортировка, сложность алгоритмов)** — базовые сортировки + оценка сложности
- **Лаба 4 (сортировки 2)** — дополнительные сортировки / улучшения
- **Лаба 5 (поиск, лабиринт)** — алгоритмы поиска, задачи типа “лабиринт”
- **Лаба 6 (поиск строк)** — алгоритмы поиска подстроки / работа со строками
- **Лаба 8 (жадные алгоритмы)** — задачи на greedy-стратегии
- **Лаба 9 (динамическое программирование)** — задачи на DP
- **Лаба 10 (ДЗ)** — домашние задания / дополнительные задачи

> Названия папок сохранены в исходном виде (как в учебном курсе).
